Interactive Page

Enter the side-length of the grid. Number must be between 2 and 50, inclusive. Hit the redraw button to reset the grid to its last size. Click and drag to mark walls, or remove walls. Hit Solve to find the least cost path from the top left to the bottom right. The solution is derived using the A* algorithm and avoids any user created barriers.

Enter one number, it will be used as both Length and Width


Hit the snake button to play snake. Snake uses the WASD keys for movemont, and ends the game if you touch the edges. If you would like to wraparound the screen when hitting the edges, hit the wraparound button. Hitting the snake button again also pauses/plays the game.
